# IT Support Tickets

## Overview

This project documents hands-on desktop support and service desk scenarios completed through an IT Support Simulator. The lab focuses on troubleshooting common enterprise IT issues involving Active Directory, user account management, network connectivity, VPN access, email services, server administration, MFA, and infrastructure support.

The objective of this lab was to practice structured troubleshooting, incident management, root cause analysis, user communication, and ticket documentation in a simulated corporate environment.

## Example Troubleshooting Workflow

1. Review ticket details and business impact
2. Identify affected user(s) or systems
3. Verify user identity when required
4. Investigate root cause
5. Implement corrective action
6. Validate service restoration
7. Confirm resolution with user
8. Document findings and resolution
