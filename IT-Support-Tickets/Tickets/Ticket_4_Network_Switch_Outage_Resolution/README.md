# Network Switch Outage Resolution

## Ticket Summary

Multiple users on Floor 3 reported loss of internet and network connectivity. Users on other floors remained unaffected, indicating that the issue was isolated to a specific section of the network infrastructure.

## Symptoms

* Users on Floor 3 unable to access the network
* Internet connectivity unavailable
* Multiple users affected simultaneously
* Other floors operating normally

## Investigation

1. Reviewed the scope of the outage.
2. Determined that only Floor 3 users were impacted.
3. Accessed network infrastructure management tools.
4. Inspected network equipment associated with the affected floor.
5. Identified the Floor 3 network switch as the likely point of failure.

## Root Cause

The Floor 3 network switch became unresponsive, causing loss of connectivity for devices connected through that switch.

## Resolution

1. Accessed the network equipment management console.
2. Located the Floor 3 network switch.
3. Performed a controlled reboot of the affected switch.
4. Monitored the device during startup and verified it returned to an operational state.
5. Confirmed restoration of network services.
6. Verified with the user that internet and network access had been restored.

## Resolution Notes

Investigated reports of network connectivity issues affecting users on Floor 3. Determined that the outage was isolated to a single floor, indicating a network infrastructure issue rather than a company-wide service disruption.

Accessed network equipment management tools and identified the Floor 3 switch as unresponsive. Performed a reboot of the affected switch and monitored the device until normal operation was restored. Confirmed with the user that network connectivity and internet access were functioning correctly following the reboot.

Issue resolved and ticket closed.

## Key Learning Points

* Verify the scope of an outage before troubleshooting.
* Distinguish between user-specific and infrastructure-wide issues.
* Identify affected network segments to narrow down root cause.
* Network switches can become unresponsive due to software or hardware-related issues.
* Infrastructure failures often affect multiple users simultaneously.
