# New User Account Provisioning and Access Assignment

## Ticket Summary

A new employee joined the Engineering department and required access to company systems. The onboarding request included creation of an Active Directory account and assignment of department-specific access groups.

## Symptoms

* New employee unable to access company resources
* No Active Directory account existed
* Required access groups not yet assigned
* User onboarding request submitted by management

## Investigation

1. Reviewed the onboarding request and required access permissions.
2. Verified the employee details provided in the ticket.
3. Identified the required security and access groups.
4. Confirmed the requested access aligned with the employee's role.

## Root Cause

The employee was newly hired and had not yet been provisioned with an Active Directory account or required group memberships.

## Resolution

1. Created a new Active Directory user account for the employee.
2. Verified successful account creation.
3. Added the user to the Engineering group.
4. Added the user to the VPN-Users group.
5. Added the user to the GitHub-Access group.
6. Verified all required group memberships were assigned correctly.
7. Confirmed the account was ready for use.

## Resolution Notes

Processed a new hire onboarding request by creating an Active Directory account for the employee. Reviewed the requested access requirements and assigned the user to the Engineering, VPN-Users, and GitHub-Access groups in accordance with the approved access request.

Verified successful account creation and group membership assignments to ensure the user had appropriate access to departmental resources and company systems.

Issue resolved and onboarding completed successfully.

## Key Learning Points

* New hire onboarding requires both account creation and access provisioning.
* Users should only receive access specifically requested and approved.
* Security group membership determines resource access across the organization.
* Proper onboarding ensures users can be productive on their first day while maintaining security controls.
